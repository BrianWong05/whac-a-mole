# Whac-A-Mole Training / 手眼協調大冒險

![Hand-Eye Coordination Adventure Banner](public/mole-icon.png)

## 使命與「為什麼存在」

**手眼協調大冒險** 是一個專為 **職能治療 (Occupational Therapy)** 和 **特教教育 (Special Education)** 設計的專業輔導工具，特別針對 **特殊需求兒童 (Exceptional Children)** 量身打造。

這不僅僅是一個遊戲，更是一個實效的訓練平台，旨在：
- **訓練手部肌肉：** 鼓勵精確的動作並強化手指與手部的協調能力。
- **提升專注力：** 在受控的環境中挑戰兒童追蹤移動目標的能力。
- **「無失敗」環境：** 透過正向回饋減少焦慮並建立自信心。

### 無障礙功能亮點
- **「寬容判定」 (Antigravity 機制)：** 更大的感應範圍，協助精細動作控制力較弱的兒童輕鬆參與。
- **高對比視覺設計：** 使用清晰、鮮豔的目標與背景，輔助視覺追蹤訓練。
- **多樣化方向適應：** 完美支援橫屏與豎屏切換，適應不同設備的使用需求。

## 核心功能

### 📱 雙向螢幕支援
應用程式完全響應式設計，並針對以下情境優化：
- **直放 (手機)：** 適合單手練習與快速訓練。
- **橫放 (平板/分屏)：** 提供更大的互動區域，適合正式的治療課程。

### ⚙️ 適應性難度調整
- **簡單模式：** 每次僅出現一個目標，讓使用者專注於單一點位。
- **挑戰模式：** 增加複雜度，同時可能出現最多兩個目標。

### ✨ 正向鼓勵機制
- **正向回饋：** 每次擊中皆有高質量的音效與獎勵動畫。
- **無負面壓力：** 我們刻意不加入刺耳的錯誤音效或「遊戲結束」的處罰，以維持友善的支持氣氛。

### 🛠️ 客製化設定
- **可調速度：** 針對初學者放慢速度，或為進階訓練加速。
- **目標大小：** 可縮放互動元素的大小，以匹配兒童的運動機能。

## 技術棧

- **框架：** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **樣式：** [Vanilla CSS](https://developer.mozilla.org/en-US/docs/Web/CSS) (搭配 Tailwind CSS 實作)
- **行動端整合：** [Capacitor](https://capacitorjs.com/) (用於 Android APK 生成與原生功能調用)

---

## 開發環境設置 (如何運行)

請按照以下步驟在本地啟動專案：

1. **安裝依賴：**
   ```bash
   npm install
   ```

2. **啟動開發伺服器：**
   ```bash
   npm run dev
   ```

3. **於 Android Studio 開啟 (需先安裝 Capacitor)：**
   ```bash
   npx cap open android
   ```
