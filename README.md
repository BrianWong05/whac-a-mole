## Mission & "Why This Exists"

The **Whac-A-Mole Training** project is a specialized therapeutic tool designed for **Occupational Therapy** and **Special Education**, specifically tailored for **Exceptional Children** (Special Needs). 

Beyond being just a game, this app serves as a practical training platform to:
- **Train Hand Muscles:** Encouraging precise motor movements and strengthening finger-hand coordination.
- **Improve Focus:** Challenging children to track moving targets in a controlled environment.
- **"No-Fail" Environment:** Designed to reduce anxiety and build confidence through positive reinforcement.

### Accessibility Features
- **"Forgiving Hitboxes" (Antigravity Mechanic):** Larger interaction zones to assist children with limited fine motor control.
- **High Contrast Visuals:** Clear, vibrant targets and backgrounds to aid visual tracking.
- **Adaptable Orientation:** Seamlessly transitions between Landscape and Portrait modes to suit different device setups.

## Local Development (How to Run)

To set up the project locally, follow these steps:

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Development Server:**
   ```bash
   npm run dev
   ```

3. **Open in Android Studio (requires Capacitor setup):**
   ```bash
   npx cap open android
   ```
